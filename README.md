# Twitter-Bot

Create a twitter developer account and fill details in keys.py<br>

# Linux
In Terminal
```
git clone https://github.com/vineelsai26/Twitter-Bot.git
cd Twitter-Bot
pip3 install tweepy
python3 Bot.py
```

# Windows
In CMD
```
git clone https://github.com/vineelsai26/Twitter-Bot.git
cd Twitter-Bot
pip install tweepy
python Bot.py
```
